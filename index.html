My Work

This is a small portfolio of my favorite works, which were mostly done outside of school. While I have created many more projects and applications such as Pong, Blackjack, and Tron, I felt these best showcase the skills I've learned. The projects are available to download but you must have Java installed on your computer for them to run. Thanks for reading and enjoy!


Falling Pizza

Falling Pizza, while not as impressive as some of my other games, is the first game I ever programmed. The game is not very impressive, the artwork is sloppy, and there are some major flaws. However, this game was my first experience with Java graphics and game programming and gave me my first taste of what I am capable of creating.

Castle Attack

A fun, addictive single-player defense game. You must spawn soldiers and fire projectiles to try to destroy the enemy's castle before he destroys yours! Use gold to purchase upgrades and special items in the shop. In case you want to pick up where you left off, the game features save and load capabilities. Working on this project taught me about both writing to and reading from XML files in Java along with creating my own custom GUI.
(February 2014 - July 2014)

Download

Battleship

A classic game for everyone: Battleship! Be the first to figure out where the enemy has hidden their ships. Play either single-player mode against a CPU or play against a friend via LAN. This was my first major project and took around a year and a half to complete. Before starting Battleship, I knew little to nothing about making GUIs in Java and absolutely nothing about network I/O, so this project was a fun way to learn new skills.
(November 2012 - January 2014)

Bookler

My current project, Bookler, is a hybrid web app used to help teachers manage their class libraries. This app is written in HTML, CSS, and JavaScript and will be available on both Android and iOS app stores.

Air Sync

Another current project, Air Sync, will let Android users sync their music libraries wirelessly with their computers. I am nearing completion with the back end (connecting client and server, sync algorithms) and working on user interface.
